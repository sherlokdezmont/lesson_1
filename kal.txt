Питон это Тьюринг полный интерпретируемый язык программирования

Типы данных:

Числовые типы данных - ind, float, complex

Тип данных последовательности - list, tuple, range

Тип данных для передачи строковых значений - string

Двоичный тип данных - bits, bytearray, memoryview

Тип данных для сопоставления - dict (Хранит в себе наборы в виде ключа и значения - key-value)

Логический тип данных - bool (может в себе содержать только два логических значения True/ False)

Тип данных множества - set, frozenset (Используется для упорядочивания уникальных элементов)
